# Project Documentation

## Features
- Comprehensive news aggregation from various tech sources.
- User-friendly interface for easy navigation.
- Filtering options based on categories and topics.
- Push notifications for breaking news.

## Installation Instructions
1. Clone the repository:
   ```
   git clone https://github.com/shawnzhsh/daily-tech-news.git
   ```
2. Navigate to the project directory:
   ```
   cd daily-tech-news
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the application:
   ```
   npm start
   ```

## Project Structure
- `src/`: Contains all source code files.
- `public/`: Publicly accessible files, including `index.html`.
- `tests/`: Contains test files and configurations.
